# **App Name**: Datafolio

## Core Features:

- Home Page: An overview of the data scientist's skills and experience.
- Education and Certifications: A dedicated page to list educational background, degrees, and relevant certifications with descriptions.
- Academy Projects Showcase: Display projects completed during data science training with descriptions, links to code repositories, and visual outputs, using a LLM tool to generate summaries.
- Contact Form: A form for potential employers or collaborators to reach out to the data scientist.

## Style Guidelines:

- Primary color: Navy blue (#2E3A87) to evoke trust and professionalism.
- Background color: Light gray (#F0F4F8) to provide a clean, neutral canvas.
- Accent color: Teal (#26A69A) to highlight key sections and call-to-action buttons.
- Body font: 'Inter', a sans-serif for readability and a modern feel.
- Headline font: 'Space Grotesk', a sans-serif for a contemporary, techy feel; if longer text is anticipated, use this for headlines and 'Inter' for body.
- Use clean, professional icons related to data science, such as charts, graphs, and code snippets.
- Maintain a clean and structured layout with clear sections for easy navigation.
- Subtle transitions and animations when navigating between pages or displaying project details.
